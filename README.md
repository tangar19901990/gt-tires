# GT Tires ERP

ERP система для шиномонтажу GT Tires Service · Велика Димарка

## Встановлення на телефон (PWA)

1. Відкрий посилання GitHub Pages на телефоні
2. Chrome → меню (⋮) → "Додати на головний екран"
3. Або натисни кнопку "📱 Встановити як додаток" в додатку

## Розгортання на GitHub Pages

1. Створи репозиторій на github.com
2. Завантаж всі файли: `gt-tires-v4.html`, `manifest.json`, `sw.js`, `icon-192.png`, `icon-512.png`
3. Settings → Pages → Source: main branch
4. Готово! Посилання: `https://USERNAME.github.io/REPO/gt-tires-v4.html`

## Файли
- `gt-tires-v4.html` — основний додаток
- `manifest.json` — PWA маніфест  
- `sw.js` — Service Worker (офлайн режим)
- `icon-192.png` — іконка 192×192
- `icon-512.png` — іконка 512×512
